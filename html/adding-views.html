Source: https://github.com/cmakler/kgjs

Goal: properly scale container.

Scope:

  • Define container height by aspect ratio -- e.g., 2 means height = width / 2.
  • Generate "height" attribute on container div
  • On a window resize event, update height

Approach:

Get client width of container div; set client height; listen for window resize events and do same.
